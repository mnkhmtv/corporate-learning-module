# Internal Training & Mentoring System

A simple corporate training module that connects employees with mentors and courses for skill development.

## Key Features

- **Training Requests** - Employees can request to learn new topics
- **Mentor/Course Selection** - Find available mentors or short courses by skill/department
- **Session Booking** - Apply and get assigned to training sessions
- **Feedback System** - Share simple ratings and reviews after completion
- **Personal Dashboard** - Track application status and training history

## How It Works

1. **Request** → Employee submits training request with desired topic
2. **Choose** → Select from available mentors or courses
3. **Join** → Get assigned to session/course
4. **Review** → Share feedback after completion

## Data Storage

- Employee & mentor profiles
- Training applications
- Ratings and reviews

## Benefits

- Simple and intuitive interface
- Quick development onboarding
- Transparent learning community
- Minimal role complexity

---

*MVP Focus: Core training workflow without complex competency mapping or advanced analytics.*
